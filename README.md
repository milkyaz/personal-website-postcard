# Personal Postcard

Небольшой сайт-открытка на Vue 3 + Vite.

## Локальный запуск

```bash
cd client
npm install
npm run dev
```

## Деплой на Vercel

Проект уже настроен через `vercel.json` (сборка из `client/`).

### Через UI
1. Импортируй репозиторий в Vercel.
2. Нажми `Deploy`.

### Через CLI

```bash
npm i -g vercel
vercel
vercel --prod
```
